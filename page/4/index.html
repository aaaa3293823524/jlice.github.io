Docker基础学习笔记

2018-01-27

Docker基本使用

docker search 搜索镜像

docker pull REPOSITORY[:TAG] 获取镜像

docker run [OPTIONS] IMAGE [COMMAND] [ARG...] 运行Docker容器。
（如果本地没有镜像会自动从仓库获取）

  • -d 后台运行
  • -P 容器端口随机映射到宿主机
  • -p 宿主机端口:容器端口 端口绑定
  • -i 启动一个可交互容器
  • -t 使用虚拟终端关联到容器的标准输入输出
  • -v 宿主机目录:容器目录 挂载数据卷
  • --rm 使用后销毁
